# 📖 About This Repository

This repository is a **complete hands-on Java Testing practice collection** built to master **software testing concepts** used in enterprise Java applications.

It demonstrates industry-standard testing techniques using:

- ✅ JUnit 5
- ✅ Advanced JUnit
- ✅ Mockito
- ✅ Spring Boot Testing
- ✅ Spring Mockito
- ✅ REST API Testing
- ✅ Integration Testing
- ✅ MockMvc
- ✅ SLF4J Logging

Every module is developed as an independent Maven project with practical examples, making this repository suitable for **learning, interview preparation, and showcasing Java testing skills to recruiters.**

# ▶️ Getting Started

## Clone Repository

```bash
git clone https://github.com/<your-username>/JUnit-Practice.git
```

---

## Navigate to Project

Example

```bash
cd JUnit-Practice/JUnitPractice
```

or

```bash
cd JUnit-Practice/Mockito-Practice
```

or

```bash
cd JUnit-Practice/Spring-Testing
```

---

## Build Project

```bash
mvn clean install
```

---

## Run Tests

```bash
mvn test
```
